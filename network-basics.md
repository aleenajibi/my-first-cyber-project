# What Happens When You Visit google.com?

1. You type google.com in the browser.
2. Your computer asks a DNS server for the IP address.
3. DNS replies with the IP address.
4. Your computer sends a TCP SYN request.
5. The server replies with SYN-ACK.
6. Your computer sends ACK.
7. HTTPS secure connection is created.
8. The website loads.

## What I Learned
- DNS converts domain names into IP addresses.
- TCP uses a 3-way handshake.
- HTTPS encrypts data.
